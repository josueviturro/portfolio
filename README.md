# 🚀 Portfolio Personal - Josué Viturro

Bienvenido al repositorio de mi portfolio web. Este sitio interactivo fue diseñado y desarrollado desde cero para exhibir mis proyectos de software, mi stack tecnológico y mi enfoque para resolver problemas mediante código limpio.

## 💻 Tecnologías Utilizadas en este Proyecto

Este sitio fue construido utilizando herramientas modernas de alto rendimiento:
- **React:** Para la construcción de interfaces de usuario basadas en componentes.
- **TypeScript:** Para aportar tipado estático, reduciendo errores y mejorando la escalabilidad del código.
- **Vite:** Como entorno de desarrollo y empaquetador ultrarrápido.
- **HTML5 & CSS3:** Para una maquetación responsiva y un diseño profesional.

## 📂 ¿Qué vas a encontrar en mi web?

Mi portfolio destaca mis desarrollos principales, enfocados tanto en el entorno de escritorio como en la web:
1. **Alpha Gestión:** Un sistema de administración integral para PyMEs desarrollado en **C# y .NET MAUI**, con persistencia local de datos y generación automatizada de reportes (Excel/PDF).
2. **MisiónSeg:** Una landing page corporativa moderna y funcional para una empresa de seguridad privada.
